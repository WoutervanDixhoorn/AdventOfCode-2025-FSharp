## How to run the solutions
These solutions are built to take input via the command line pipe.

### Running the code
1. Navigate to the specific day's directory (e.g., `cd Day01`).
2. Run the following command based on your OS:

**Windows (PowerShell)**
```powershell
Get-Content input.txt > dotnet fsi main.fsx
```

**Linux (Bash)**
```bash
cat input.txt | dotnet fsi main.fsx
```
